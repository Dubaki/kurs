# Структура обновленного PDF-отчета для клиента (Lead Magnet)

Этот отчет должен быть персонализированным ответом на прохождение Квиза. Основная цель: подтвердить экспертность и закрыть на встречу/демо.

## 1. Обложка (Персонализация)
* **Заголовок:** «План внедрения ИИ-агента для повышения рентабельности бизнеса».
* **Подзаголовок:** «Специализированный отчет для ниши: {НИША_ИЗ_КВИЗА}».
* **Визуал:** Стилизованный робот (как на сайте) в окружении иконок выбранной индустрии.

## 2. Проблема и решение (на основе "Боли" из Квиза)
* Если выбрана «Потеря лидов»: график "Воронка без ИИ vs Воронка с ИИ". Показываем, как 24/7 обработка в WhatsApp закрывает "дыру" в маркетинге.
* Если выбрана «Ошибки/Рутина»: Сравнение стоимости 1 часа работы человека и ИИ-агента (на основе данных из 7-й ниши PDF).

## 3. Математика вашего профита (Важнейший блок)
* Используем данные из Квиза (Зарплата, Кол-во сотрудников).
* **Формула:** (Экономия ФОТ) + (Доп. выручка от спасенных лидов) - (Стоимость внедрения).
* **Срок окупаемости:** Акцент на «ROI от 3 месяцев», как указано в исследовании.

## 4. Как это работает (Архитектура)
* Схема из `Analiz.pdf` (Интеллектуальное ядро -> CRM -> Мессенджеры).
* Описание безопасности: «Ваши данные не покидают контур компании» (особенно важно для ниши Юристы/IT).

## 5. Кейсы (мини-формат)
* Вставить по 1 краткому примеру на каждую нишу из `cases.html`. Например: «Для медцентра: +14% к записи за первый месяц».

## 6. Призыв к действию (CTA)
* Не просто «Позвоните нам», а «Получить индивидуальную архитектуру решения».
* Ссылка на Telegram: [@tonmeplz](https://t.me/tonmeplz).
* QR-код на запись в календарь.
