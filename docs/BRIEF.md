# BRIEF: Описание проекта 66ai.ru

## 1. Суть
Платформа обучения Vibe Coding и ИИ-Аттестации для бизнеса. Позволяет владельцам компаний самостоятельно автоматизировать процессы через нейросети, без найма программистов.

## 2. Целевая аудитория
| Сегмент | Описание | Ключевая боль |
|---|---|---|
| Владельцы МСБ | Принятые решения об автоматизации и бюджете | Раздутый ФОТ, нехватка IT-кадров |
| HR-директора | Оценка и развитие компетенций сотрудников | Невозможно измерить реальный уровень ИИ-навыков |
| Эксперты-одиночки | Юристы, бухгалтеры, налоговики, психологи | Большие массивы данных, рутина, нет IT-поддержки |
| Смежные ниши | Маркетинг, техподдержка, B2B-продажи, логистика | Страх отстать от конкурентов, уже внедривших ИИ |

## 3. Scope (Входит в проект)
- Лендинг-презентация курсов Vibe Coding (многостраничный HTML-сайт)
- Страница B2B-услуги «Аттестация 66AI»
- Telegram-бот для многоуровневого тестирования сотрудников
- HR-Дашборд с аналитикой и бенчмаркингом
- PDF-сертификация с QR-кодом верификации
- Публичная страница верификации `/verify/{ID}`

## 4. Out of Scope (Не входит)
- Создание сложной LMS-системы с нуля
- Мобильные приложения iOS/Android (нативные)
- Интеграция с 1С, SAP или другими ERP-системами заказчика
- Хостинг и развёртывание на сервере заказчика

## 5. Технологический стек
| Слой | Технология |
|---|---|
| Фронтенд | HTML5, Tailwind CSS (CDN, без сборки), Google Fonts |
| Бэкенд | PHP (скрипты в `/api/`) |
| Telegram-бот | Python + aiogram (планируется) |
| Хранение | Supabase / PostgreSQL (планируется) |
| PDF-генерация | WeasyPrint / ReportLab (планируется) |
| Хостинг | Статический деплой (GitHub Pages / свой сервер) |

## 6. Структура данных (проектная)
| Сущность | Поля |
|---|---|
| Пользователь (HR/Руководитель) | ID, ФИО, email, компания, отрасль, роль |
| Сотрудник (Тестируемый) | ID, ФИО, email, привязка к руководителю |
| Результат теста | ID, ID_сотрудника, уровень (1-3), баллы по компетенциям, дата, UUID_сертификата |
| Сертификат | UUID, ID_результата, QR-ссылка, хеш_подписи, дата_выдачи |
