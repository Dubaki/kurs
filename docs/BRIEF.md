# BRIEF: Описание проекта 66ai.ru

> ⚠️ Обновлено: 13.04.2026. Отражает старт разработки Telegram Mini App для тестирования сотрудников и завершение глобального рефакторинга UI (Сплит-Шлюз).

## 1. Суть
Единая B2B-платформа ИИ-трансформации, объединяющая диагностику компетенций (Аттестация 66AI), базовое обучение продуктивности (курс «Убийца Рутины») и продвинутую академию создания IT-решений без кода (VibeCraft). Платформа решает проблему низкой эффективности офисного труда и помогает бизнесу безопасно внедрять нейросети.

## 2. Целевая аудитория
| Сегмент | Описание | Ключевая боль |
|---|---|---|
| Владельцы МСБ | Принимают решения об автоматизации и бюджете | Раздутый ФОТ, нехватка IT-кадров, страх отстать от рынка |
| HR-директора | Оценка и развитие компетенций команды | Невозможность измерить реальный уровень ИИ-навыков сотрудников |
| Эксперты-одиночки | Юристы, бухгалтеры, аудиторы, аналитики | Большие массивы данных, рутина, отсутствие IT-поддержки |
| Предприниматели | Созидатели, желающие создавать свои IT-продукты | Дорогая и долгая классическая разработка |

## 3. Scope (Входит в проект)
- **Фасад-шлюз (Gateway):** Интерактивная главная страница с 3D-параллакс карточками для разделения трафика на «Аттестацию» и «VibeCraft».
- **Аттестация 66AI (Telegram Mini App):** Автоматизированное многоуровневое тестирование сотрудников через встроенное веб-приложение (TMA) внутри Telegram.
- **Базовый курс «Убийца Рутины»:** Обучающая программа по безопасности и базовой ИИ-грамотности (маршрутизация для сотрудников с низким баллом).
- **Академия VibeCraft:** Обучение продвинутой методологии создания IT-решений (маршрутизация для сотрудников с высоким баллом).
- **B2B Аудит:** Бесплатная диагностика процессов для компаний (лид-магнит при аттестации от 10 сотрудников).
- **HR-Дашборд:** Панель руководителя с AI Skill Radar (радарная диаграмма) и бенчмаркингом.
- **Верификация:** Генерация PDF-сертификатов с QR-кодом и публичная страница проверки `/verify/{ID}`.

## 4. Out of Scope (Не входит)
- Создание сложной LMS-системы с нуля (используем готовые решения или простые форматы).
- Разработка собственных больших языковых моделей (LLM).
- Мобильные приложения iOS/Android (нативные).
- Сложные интеграции с 1С, SAP или внутренними ERP заказчика.
- Хостинг и развёртывание на серверах заказчика (on-premise).

## 5. Технологический стек
| Слой | Технология |
|---|---|
| Фронтенд (Сайт) | HTML5, Tailwind CSS (CDN, без сборки), Vanilla JS |
| Фронтенд (TMA) | HTML5, Tailwind CSS, Telegram Web Apps API |
| Бэкенд | PHP (базовые API) / Python FastAPI (планируется для ML) |
| Telegram-бот | Python + aiogram (планируется) |
| Хранение | Supabase / PostgreSQL (планируется) |
| PDF-генерация | WeasyPrint / ReportLab (планируется) |
| Аналитика | Chart.js (для радарных диаграмм) |
| Хостинг | Статический деплой |

## 6. Структура данных (проектная)
| Сущность | Поля |
|---|---|
| Пользователь (B2B) | ID, ФИО, email, компания, отрасль, роль |
| Сотрудник | ID, ФИО, email, привязка к B2B-пользователю |
| Результат теста | ID, ID_сотрудника, баллы по компетенциям, рекомендации (курс) |
| Сертификат | UUID, ID_результата, QR-ссылка, хеш_подписи, дата_выдачи |
