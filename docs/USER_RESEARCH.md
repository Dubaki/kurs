# Исследование ЦА и Дизайн-код проекта 66ai.ru

## 1. Целевая аудитория

- **Владельцы малого и среднего бизнеса, HR-директора.** Люди, принимающие решения об автоматизации и обучении команды.
- **Эксперты-одиночки с массивами данных:** Юристы, Бухгалтеры, Налоговики, Психологи. Специалисты, которым нужно обрабатывать большие объёмы информации и документов без помощи IT-отдела.
- **Смежные ниши:** Маркетинг, Техподдержка, B2B-продажи, Логистика. Отделы, где рутинные операции занимают значительную часть рабочего времени.

## 2. Ключевые боли и Мотивация

- **Боль:** Страх отстать от конкурентов, которые уже внедрили ИИ. Ощущение, что «поезд уходит», но нет понятного пути входа.
- **Боль:** Раздутый ФОТ (фонд оплаты труда) и низкая эффективность текущих сотрудников на рутинных задачах. Найм новых специалистов дорог и долог.
- **Мотивация:** Оптимизировать процессы, повысить выработку текущей команды за счёт ИИ — без найма новых людей. Получить конкретные инструменты, а не «теорию об ИИ».

## 3. Визуальный стиль (Дизайн-код)

- **Концепция:** B2B Premium / Strict Minimalism. «Дорого, строго, только суть». Минимум декора — максимум смысла.
- **Цветовая палитра:** Глубокий графит (Slate) для фона, матовый белый для контраста. Акценты: Изумрудный (рост, деньги, ассоциация с текущими ассетами проекта) или Индиго (технологичность).
- **Табу:** Никакого «AI-слопа» — фиолетовых светящихся нейросетей, дешёвых градиентов. Запрет на чистый чёрный (`#000000`) и чистый белый (`#FFFFFF`) — только приглушённые оттенки.
- **Элементы:** Glassmorphism (только для акцентных карточек), строгая типографика (Space Grotesk + Manrope), много воздуха (white space), чёткая иерархия контента.
