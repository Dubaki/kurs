# Структура проекта: Сайт «Дядя Фёдор»

## Иерархия страниц
- `index.html`: Главная. Оффер (ИИ без хайпа. Только польза), основные услуги, CTA (Бесплатный аудит).
- `about.html`: История и команда. Фокус на экспертности (г. Екатеринбург, Шахмаев А.О.) и простом подходе к сложным технологиям.
- `cases.html`: Примеры пользы. Визуальный каталог решений для разного бизнеса (Экономия API, Малый бизнес, Продажи, Сервис, HR, Производство). 
- `style.css`: Единый визуальный стиль (Брутализм, контрастный красный, Montserrat).

## Контентная стратегия Блога
Каждая статья должна быть подробным руководством по извлечению прибыли из ИИ. 

### Шаблон статьи:
1. **Проблема:** Какую боль бизнеса мы решаем? (Например: "Операторы тратят 3 часа в день на ответы в WhatsApp").
2. **Решение:** Как именно ИИ-агент может помочь? (Технический стек без лишних деталей: LLM, интеграция с CRM).
3. **Экономика:** Подробный расчет прибыли. 
    *   *Пример:* Стоимость внедрения X руб. -> Экономия Y часов в месяц -> Окупаемость Z месяцев.
4. **Масштабируемость:** Можно ли это применить в других сферах?

### Темы первых постов:
- "Как ИИ-агент за 5000 рублей в месяц заменяет ночного администратора".
- "Автоматизация 1С через голосовые команды: экономим время бухгалтера".
- "Почему локальная Llama 3 выгоднее подписки на ChatGPT для компании от 10 человек".

## Технические особенности
- Статический HTML/CSS для максимальной скорости загрузки.
- Использование "шумовых" текстур и SVG-анимаций для визуального богатства без потери производительности.
- Мобильная адаптивность через Flexbox и Media Queries.
