# Сайт «Курс ИИ для бизнеса»

Профессиональный лендинг с интеграцией Telegram-бота для сбора заявок.

## Быстрый старт

### 1. Настройка Telegram-бота
Чтобы формы на сайте отправляли заявки в ваш Telegram, выполните следующие действия:

1. Перейдите в папку `api/`.
2. Создайте копию файла `config.php.example` и назовите её `config.php`.
3. Откройте `config.php` и вставьте ваш **Bot Token** (от @BotFather) и **Chat ID**.
4. Сохраните файл.

*Примечание: Файл `config.php` автоматически игнорируется Git и никогда не попадет в публичный репозиторий для вашей безопасности.*

### 2. Развертывание
Сайт написан на чистом HTML/CSS/JS с небольшим PHP-бэкендом для отправки заявок.
Для работы форм сервер должен поддерживать **PHP 7.4+**.

### 3. Структура проекта
- `index.html`, `about.html`, `cases.html` — основные страницы.
- `api/send-lead.php` — обработчик форм.
- `robots.txt`, `sitemap.xml` — SEO-настройки.

## Безопасность
Проект проверен на наличие открытых API-ключей. Все секретные данные вынесены в конфигурационные файлы, исключенные из системы контроля версий.
