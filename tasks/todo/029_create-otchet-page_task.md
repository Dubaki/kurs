<task>
1. Создать файл `otchet.html`. Использовать базовую разметку проекта (взять `<head>` и `<header>` из `attestation.html` с отключенным меню).
2. Создать секцию с 3 крупными кнопками/карточками:
   - "Пример отчёта компании"
   - "Пример отчёта руководителя"
   - "Пример отчёта сотрудника"
3. Реализовать логику просмотра PDF (интеграция файлов из папки `docs/`):
   - ВАРИАНТ А (Модальное окно): При клике на карточку открывается модальное окно на весь экран с тегом `<object data="docs/...pdf" type="application/pdf" class="w-full h-full">` или `<iframe>`.
   - ВАРИАНТ Б (Аккордеон/Слайдер): При клике блок разворачивается вниз, загружая PDF внутри себя.
   - Использовать Вариант А (модальное окно) как более предпочтительный для чтения документов.
4. Добавить кнопку "Назад к Аттестации" (ведущую обратно на `attestation.html`).
5. ВЕРИФИКАЦИЯ:
   - Открыть `otchet.html`.
   - Кликнуть на "Пример отчёта компании" -> должен отобразиться соответствующий PDF из папки docs.
   - Закрыть модальное окно, кликнуть на другие.
6. Заполнить COMPLETION LOG и перенести в done/.
</task>
